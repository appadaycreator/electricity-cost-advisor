# 電気代節約診断【無料】月々いくら安くなる？最適プランを提案 - 技術仕様書

## 概要

**サービス名**: Electricity Cost Advisor
**バージョン**: 1.5.0（2026-06-11 M6目標トラッキング実装）
**更新日**: 2026-06-11
**URL**: https://appadaycreator.com/electricity-cost-advisor/

家族構成・使用量を入力するだけで電気代を節約できる最適プランを無料診断。新電力・電力会社比較。

## バージョン履歴

**v1.5.0 (2026-06-11) - P1・P2改善実装（M6目標トラッキング）**
- M6実装: 削減額の目標設定機能を追加（目標額入力 → 達成度プログレスバー表示）
- 目標データを localStorage に保存し、再訪問時に自動復元
- P2実装: 前回診断結果の自動復元・表示機能を追加

  - 「前回結果を見る」ボタンで即座に詳細確認可能
  - 「前回値で再診断」ボタンで前回条件を自動入力して再診断開始
- 目標達成度を達成率パーセンテージで表示
- 達成度に応じた段階的メッセージ表示（未設定 → 開始段階 → 進行中 → あと少し → 目標達成）

**v1.4.0 (2026-06-10) - P1・P2改善実装**

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 電気代節約診断は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
